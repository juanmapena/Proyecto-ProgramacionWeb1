JavaScript

Valor curso: $150

Tiempo de dedicación necesario: 12 semanas

Descripción del curso: Aprende a programar en JavaScript, el lenguaje esencial para el desarrollo web, con proyectos prácticos y dinámicos.

Requisitos previos: Conocimientos básicos de informática.

CONTENIDOS POR CLASE

Introducción

📘 Qué es JavaScript y su rol en el desarrollo web.

🎥 Video: “Cómo se conecta HTML, CSS y JS”.

💻 Actividad: primer script con console.log().

🧩 Test inicial de conceptos básicos.

Unidad 2: Variables y Tipos de Datos

📘 Uso de let, const y var.

🎥 Video: “Buenas prácticas con variables”.

💻 Ejercicio: programa de cálculos simples con entrada del usuario.

📄 PDF: “Tipos de datos primitivos en JS”.

Unidad 3: Operadores y Condicionales

📘 Operadores lógicos, relacionales y aritméticos.

🎥 Video: “Flujo de decisiones con if, else if y else”.

💻 Ejercicio práctico: simulador de descuentos según edad.

🧠 Cuestionario de repaso.

Unidad 4: Bucles y Arrays

📘 Estructuras repetitivas for, while, for...of.

🎥 Video: “Cómo recorrer y manipular arreglos”.

💻 Desafío: crear una lista dinámica de tareas (ToDo List).

🧩 Examen parcial de unidad.

Unidad 5: Funciones y Eventos

📘 Declaración y expresión de funciones.

🎥 Video: “Eventos y manipulación del DOM”.

💻 Ejercicio: crear un contador con botones + y -.

📄 PDF: “Guía de funciones flecha”.

Unidad 6: Proyecto Final

🧩 Proyecto integrador: desarrollo de una página interactiva con DOM y eventos.

💡 Aplicación de funciones, arrays y condicionales.

📤 Entrega final y revisión con feedback del docente.

🏅 Certificación de finalización.
