Java

Valor curso: $110

Tiempo de dedicación necesario: 10 semanas

Descripción del curso: Aprende a programar en Java, uno de los lenguajes más utilizados para aplicaciones empresariales, móviles y de escritorio. Desarrolla proyectos prácticos para dominar sus conceptos clave.

Requisitos previos: Conocimientos básicos de programación.

CONTENIDOS POR CLASE

Introducción

📘 Historia y características de Java.

🎥 Video: “Por qué Java sigue siendo uno de los lenguajes más usados”.

🧩 Instalación de JDK y configuración del entorno (Eclipse o IntelliJ).

💻 Primer programa: “Hola Mundo”.

Unidad 2: Tipos de Datos y Variables

📘 Tipos primitivos, variables y constantes.

🎥 Video: “Cómo se almacenan los datos en memoria”.

💻 Ejercicio: programa que calcule área y perímetro.

📄 PDF: “Guía de tipos primitivos en Java”.

Unidad 3: Estructuras de Control

📘 Condicionales if, else, switch.

💻 Ejercicio práctico: simulador de notas y aprobación.

🎥 Video: “Control del flujo de un programa”.

🧠 Cuestionario de repaso.

Unidad 4: Bucles y Colecciones

📘 Uso de for, while y do-while.

💻 Ejercicio: recorrer arrays y mostrar datos por consola.

🎥 Video: “Colecciones en Java: ArrayList y HashMap”.

🧩 Evaluación práctica de unidad.

Unidad 5: Programación Orientada a Objetos

📘 Conceptos: clases, objetos, atributos y métodos.

🎥 Video: “Creación de clases y encapsulamiento”.

💻 Desafío: sistema de gestión simple con clases Alumno y Curso.

📄 PDF: “Guía rápida de OOP”.

Unidad 6: Proyecto Final

🧩 Proyecto integrador: aplicación completa en Java con OOP.

💡 Uso de arrays, métodos, clases y relaciones.

📤 Entrega final y defensa del proyecto.

🏅 Evaluación integral y certificado de finalización.
