Python

Valor curso: $120

Tiempo de dedicación necesario: 6 semanas

Descripción del curso: Aprende los fundamentos de Python con ejemplos prácticos y proyectos.

Requisitos previos: Conocimientos básicos de informática.

CONTENIDOS POR CLASE

Introducción

📘 Presentación del curso y objetivos.

🎥 Video: “Qué es Python y para qué sirve”.

📝 Actividad: Instalación de Python y primer script.

🧩 Examen inicial de conocimientos.

Unidad 2: Sintaxis y Variables

📘 Conceptos: tipos de datos, variables y operadores.

🎥 Video: “Cómo funciona la asignación en Python”.

📄 PDF de apoyo: “Resumen de tipos de datos”.

💻 Práctica: crear un programa que calcule el promedio de notas.

🧠 Cuestionario de repaso.

Unidad 3: Estructuras de Control

📘 Teoría: condicionales if, elif, else.

🎥 Video: “Lógica condicional aplicada a ejemplos reales”.

💻 Ejercicio guiado: sistema de aprobación de estudiantes.

🧩 Examen parcial de la unidad.

Unidad 4: Bucles y Colecciones

📘 Contenido: for, while, listas, tuplas y diccionarios.

🎥 Video: “Recorriendo listas y procesando datos”.

💻 Desafío: crear un programa que simule un carrito de compras.

🧠 Mini test interactivo.

Unidad 5: Funciones y Módulos

📘 Definición y utilidad de funciones.

💻 Ejercicio: dividir un programa grande en funciones reutilizables.

🎥 Video: “Parámetros, retorno y buenas prácticas”.

📄 PDF: “Guía de funciones predefinidas”.

Unidad 6: Proyecto Final

🧩 Proyecto integrador: desarrollo de una calculadora o sistema de notas.

💡 El alumno debe aplicar todo lo aprendido en un caso real.

📤 Entrega y defensa del proyecto final.

🏅 Evaluación final y feedback personalizado del docente.
